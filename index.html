Segment 8 - Keybinds

This is an intresting one, atleast for me so thats why I will go into more details.

This segment explains keybinds, you are able to bind ANY keyboard key and it will execute code or a function every time the key is pressed of course except if you are typing in chat, or your in the escape menu. It also registers when the UI is closed. The way to use it AddKeybind(Title, Key, Callback), where Key is the input needed to be pressed for callback to be executed

Let's create an example, shall we?

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

local Hub = Library.New("Tutorial")

local TutorialPage = Hub:AddPage("Example Page", 10016745130)
local TutorialSection = TutorialPage:AddSection("Example Section!")
TutorialPage:AddSearchBar()

local ExampleKeybind = TutorialSection:AddKeybind("Example Keybind",Enum.Keycode.X,function(v) 
    print("Executed") 
end)

Here when the key button x is pressed, it will print 'Executed' on the console.

Returns -> Keybind - Type -> Instance

Segment 9 - Notifications

Notifications is quite intresting, you can for example warn the user that the script wont work because a condition is met.

For example, knowing the usage is Library:Notify(Title, Text, Position (UDim2), Direction(UDim2), Callback)where Position is the starting position and Direction is the end position after tweening we can do the below script.

local foo = 10
local bar = 5

if foo >= bar then Library:Notify("Condition met", "It seems that foo is bigger than bar", UDim2.new(0,0,0,0), UDim2.new(1,0,1,0), function(v)
        if v then
            print("Notification Accepted")
        else
            print("Notification Declined")
        end
    end)
end
