Segment 1

Importing and creating a window

First, to use the library we must import. Copy the below snippet and paste it into a script.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

After we import the library its really straight forward to create an empty window.

We call the Library.New() and a string as parameter for the title

for example -> local hub = Library.New(title)

with that we make a simple window that we can drag around.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

local hub = Library.New("Tutorial")

Last thing in this segment is the :Toggle()

It basically turns on / off the UI without destroying it (has a sick animation)

Here is a example usage

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

local hub = Library.New("Tutorial")
hub:Toggle()

In this case it closes the UI because it was already enabled.

Segment 2.5

In segment 2.5 we will now create sections, they hold the interactables in a organized place.


Now, to create a section it's realllly simple, we just add a local tutorialSection = tutorialPage:AddSection(title) where title is the sections title and it basically creates an empty section for us that we can put our interactables into.

The finished example code for this 1.5 segments in below

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

local hub = Library.New("Tutorial")

local tutorialPage = hub:AddPage("Example Page", 10016745130)
local tutorialSection = tutorialPage:AddSection("Example Section!")

Segment 2.9

In segment 2.9 we will learn how to make use of the search bar.

To make use of the search bar we need to have a page It only searches on the current page page:AddSearchBar() page is our local tutorialPage = hub:AddPage("Example Page", 10016745130) so to enable the search bar for our example page we will use this.

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

Segment 3 - Buttons

Now here is the interactables part, which is really boring to make a documentation on, but im crazy.


Let's begin with the button. It's really simple just like every other interactable the library has. To create a button using the previous example code we can do the following. local exampleButton = tutorialSection:AddButton(text, callback), you're probably wondering whats the callback thingus. Well is the function executed when clicking the button. Let's quickly make an example button.

local exampleButton = tutorialSection:AddButton("Testing Button", function()

Okay, to use toggles its the same as buttons, tho it takes different parameters. To create one we can really simply use this local exampleToggle = tutorialSection:AddToggle(Title, IsEnabled, Callback) now you might think what is the IsEnabled, no just kidding let's go straight into Callback. So again it takes a function() end but the functions takes a parameter v which stands for value, now this is VERY IMPORTANT it applies to dropdowns and other interactableshttps://imgur.com/a/jfIQZAG when for example trying to print or save in another bool that toggle value you wont do foo = toggle you MUST DO foo = toggle.value insteant

Let's now add a test toggle to our script :)

Segment 3 - Dropdowns

Welcome to dropdowns, they are downs that drop. No okay so this segment we are learning how to add dropdowns. It's again really simple if you understand it. So to create an empty dropdown we simply need to use :AddDropdown(), and in our code its local exampleDropdown = tutorialSection:AddDropdown(Title, Entries, Callback), now you probably are wandering what are Entries just like I did at first. It's basically a table {"Option 1", "Option 2", "Option 3"}

So, if we'd like to impliment it into our little example script its basically the code below
