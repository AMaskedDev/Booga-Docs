Introduction

BOOGA UI is a Lua-based user interface library designed for Roblox. This documentation provides an overview of its main components and how to utilize them to create custom interfaces for Roblox games.

Setup

To use BOOGA UI in your project, include the below script to initialize the library.

local Lib = loadstring(game:HttpGet("https://raw.githubusercontent.com/bbbPriv/BOOGA-UI/main/booga%20ui.lua"))()

Main Components

Library Initialization

Initializes the main library object.

Library = {}

This creates the main table for the library, which will hold all the methods and properties for the UI components.

Theme Management

Manages UI themes including colors and fonts.

Library.Theme = {
    -- Default theme settings
}

You can customize the theme by modifying the Library.Theme table. This table includes properties for various UI elements, such as background colors, text colors, and fonts.

Main Interface

Creates the main UI frame and handles layout settings.

Library:CreateMain(title)

This method initializes the main UI container. It takes one argument:

  • title: The title of the main UI frame.

Example usage:

local main = Library:CreateMain("My Game UI")

UI Elements

Tab

Creates a new tab in the interface.

main:CreateTab(name)

This method adds a new tab to the main interface. It takes one argument:

  • name: The name of the tab.

Example usage:

local tab1 = main:CreateTab("Tab 1")

Section

Adds a section within a tab.

tab:CreateSection(name)

This method creates a new section within a tab. It takes one argument:

  • name: The name of the section.

Example usage:

local section1 = tab1:CreateSection("Section 1")

Button

Adds a button element with customizable text and callback functions.

section:CreateButton(name, callback)

This method creates a button within a section. It takes two arguments:

  • name: The text displayed on the button.
  • callback: The function to be called when the button is clicked.

Example usage:

section1:CreateButton("Click Me", function()
    print("Button clicked!")
end)

Toggle

Adds a toggle switch with customizable states and callback functions.

section:CreateToggle(name, default, callback)

This method creates a toggle switch within a section. It takes three arguments:

  • name: The text displayed next to the toggle switch.
  • default: The default state of the toggle (true or false).
  • callback: The function to be called when the toggle state changes.

Example usage:

section1:CreateToggle("Toggle Me", true, function(state)
    print("Toggle state: " .. tostring(state))
end)

Slider

Adds a slider element for numerical input with min, max, and default values.

section:CreateSlider(name, min, max, default, callback)

This method creates a slider within a section. It takes five arguments:

  • name: The text displayed next to the slider.
  • min: The minimum value of the slider.
  • max: The maximum value of the slider.
  • default: The default value of the slider.
  • callback: The function to be called when the slider value changes.

Example usage:

section1:CreateSlider("Volume", 0, 100, 50, function(value)
    print("Slider value: " .. value)
end)

Dropdown

Adds a dropdown menu with selectable options.

section:CreateDropdown(name, options, callback)

This method creates a dropdown menu within a section. It takes three arguments:

  • name: The text displayed next to the dropdown menu.
  • options: A table of options for the dropdown menu.
  • callback: The function to be called when an option is selected.

Example usage:

section1:CreateDropdown("Select Option", {"Option 1", "Option 2", "Option 3"}, function(selected)
    print("Selected option: " .. selected)
end)

Textbox

Adds a textbox for user input.

section:CreateTextbox(name, default, callback)

This method creates a textbox within a section. It takes three arguments:

  • name: The text displayed next to the textbox.
  • default: The default text in the textbox.
  • callback: The function to be called when the text is changed.

Example usage:

section1:CreateTextbox("Enter Text", "Default Text", function(text)
    print("Textbox input: " .. text)
end)

Label

Adds a label for displaying static text.

section:CreateLabel(text)

This method creates a label within a section. It takes one argument:

  • text: The text to be displayed.

Example usage:

section1:CreateLabel("This is a label")

Color Picker

Adds a color picker for selecting colors.

section:CreateColorPicker(name, default, callback)

This method creates a color picker within a section. It takes three arguments:

  • name: The text displayed next to the color picker.
  • default: The default color.
  • callback: The function to be called when the color is changed.

Example usage:

section1:CreateColorPicker("Pick a Color", Color3.fromRGB(255, 0, 0), function(color)
    print("Selected color: " .. tostring(color))
end)

Keybind

Adds a keybind setting for custom key inputs.

section:CreateKeybind(name, default, callback)

This method creates a keybind setting within a section. It takes three arguments:

  • name: The text displayed next to the keybind.
  • default: The default keybind.
  • callback: The function to be called when the keybind is changed.

Example usage:

section1:CreateKeybind("Set Keybind", Enum.KeyCode.F, function(key)
    print("Selected keybind: " .. tostring(key))
end)

Utility Functions

Helper functions for internal use such as color conversions, event connections, and layout management.

-- Example utility function
function Library:ConvertColorToHex(color)
    local r, g, b = color.r * 255, color.g * 255, color.b * 255
    return string.format("#%02x%02x%02x", r, g, b)
end

Event Handling

Methods for handling user interactions with UI elements including clicks, toggles, and input changes.

-- Example event handling
function Library:OnButtonClick(button, callback)
    button.MouseButton1Click:Connect(callback)
end

Customization

Options to customize the appearance and behavior of UI elements including themes, animations, and layout settings.

-- Example customization
Library.Theme.ButtonColor = Color3.fromRGB(255, 0, 0)

You can customize the appearance of the UI by modifying the properties in the Library.Theme table.

Examples

local Library = require("booga ui.lua")

local main = Library:CreateMain("My Game UI")

local tab1 = main:CreateTab("Tab 1")
local section1 = tab1:CreateSection("Section 1")

section1:CreateButton("Click Me", function()
    print("Button clicked!")
end)

section1:CreateToggle("Toggle Me", true, function(state)
    print("Toggle state: " .. tostring(state))
end)

section1:CreateSlider("Volume", 0, 100, 50, function(value)
    print("Slider value: " .. value)
end)

section1:CreateDropdown("Select Option", {"Option 1", "Option 2", "Option 3"}, function(selected)
    print("Selected option: " .. selected)
end)
